Angular/Ionic Addon

Mobile application for purchasing rare coins and bank notes. Built with Angular and Ionic, and run on a Node.js server. Has all the functionality of the desktop application and also connects to the same Microsoft database. An API controller had to be built in the ASP.net application to accept, process, and return database information when pinged with an http request.

Bitcoin Block Explorer

Application for searching Bitcoin blocks. I combined an implementation of Bitcoin software called Bcoin with a NodeJs server configured with the Express framework. A search would run as follows; User searches, NodeJs server recieves the request, NodeJs pings the Bcoin node via http, Bcoin returns data, the NodeJs server formats a view for the user.
